Twitter Movie Success Predictor

In this project I compiled a small database of movie data from the past 4 years and ran it through a script which searches for the total number of tweets, retweets, likes, and average sentiment of the tweets regarding each of the movies.

Afterwards, with data on each movie, I use serveral machine learning models to predict the box office success based off of the trends in the tweet analytics.

You can view the Jupyter Notebooks on Github here.

Ride-Hailing Service Analysis

In this project I take open data from released from the City of Chicago and format it for use in Tableau. This interactive visualization shows the peak times of the day that ride-hailing services are used, as well as locations where rides are called throughout the day.

You can view the full Dashboard on Tableau Public here and the code on Github here.

Reader Engagement Analysis

I scraped data from Wuxiaworld.com (an online novel hosting site) of chapters, publication dates, and user comments for several novels to do a meta-analysis on user engagement.

This concluded in finding a lower rate of user engagement when multiple chapters were released in a short period of time.

Further conclusions and the Jupyter Notebook for this project can be found here.
